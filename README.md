# Inventaire IT

Inventaire IT est une application web construite avec Next.js, Prisma et Tailwind CSS pour centraliser la gestion du parc informatique.

## Fonctionnalités principales

- Vue d'ensemble temps réel des équipements, établissements et localisations.
- Gestion complète des établissements : création et modification des informations clés.
- Suivi des modèles, fabricants, catégories et statuts d'équipements.
- Visualisation des métriques essentielles (valeur, répartition par statut, supervision réseau).

## Démarrage rapide

```bash
npm install
npm run dev
```

L'application est accessible sur [http://localhost:3000](http://localhost:3000).

## Scripts utiles

- `npm run dev` : démarre l'environnement de développement.
- `npm run build` : génère la version de production.
- `npm run start` : lance le serveur en mode production.
- `npm run lint` : exécute les vérifications ESLint.

## Technologies

- Next.js 15 et React 19
- TypeScript
- Tailwind CSS 4 & shadcn/ui
- Prisma & SQLite
